This directory serves as a pointer to the processed PMF data for the slab system at z = 10 Å, resolved by effective coordination number (ECN). 

The `ecn_z_10/ecn_2` contains a full simulation setup for ECN = 2 at z = 10 Å.

The corresponding free-energy profiles are stored in:

`data/reactive_pmfs/ecn_z_10/pmf_slab_ecn_0.dat`
`data/reactive_pmfs/ecn_z_10/pmf_slab_ecn_1.dat`
`data/reactive_pmfs/ecn_z_10/pmf_slab_ecn_2.dat`
`data/reactive_pmfs/ecn_z_10/pmf_slab_ecn_3.dat` 
`data/reactive_pmfs/ecn_z_10/pmf_slab_ecn_4.dat`
